Your Chromebook must not be too fast (but also not too slow) to perform this exploit.

How to Deservice

1. Immediately after logging in, press Search+Escape a few times. A window should appear. If the window immediately disappears, press Search+Escape a few times again.

Note that if the "End task" button is grayed out, then unfortunately this exploit will not work for you.


2. Click "All tasks" on the top bar (skip this step if you're not using the latest task manager).


3. Click "Process ID". Network Service should appear third, below GPU Process, or somewhere near it. Repeat this step if the processes are sorted the wrong way.


4. Click Network Service. It should be highlighted.


5. Press Tab. The "End task" button should be highlighted.


6. Immediately start spamming Enter and Space for around 15 seconds.


7. You can now browse Chrome without any classroom extensions interrupting your browsing! They may appear as downloaded, but they aren't actually running (assuming you've done all the steps right or the extensions did not revive themselves at some point).


8 (Optional, but is greatly recommended). Side-screen Chrome and the task manager for around a minute, then hunt for and kill any processes named "Data Decoder Service" and "Unzip Service" to prevent more extensions from loading. If you do follow this step, don't hesitate to open up a few tabs and browse a little.

If the task manager is unresponsive at any point, press Search+Escape.


To reload all extensions, restart your Chromebook or sign out and sign back in.
