Worm 🐛

Worm 🐛 is a TypeScript compiler and framework which compiles TypeScript into highly optimized and efficient machine code, compared to other runtime solutions which merely shove a JavaScript engine into your executable.

Please note that Worm 🐛 does not provide its own runtime. Use the command worm run (defaults to index.ts) to build and run your app instantly.


Worm 🐛 aims to provide:

- server, CLI, audio, and system GUI utilities with Bun-like syntax out of the box to allow TypeScript and/or Bun developers to comfortably create their own apps with Worm 🐛

- Node.js and Bun compatibility for transitioning developers

- a proprietary TypeScript lexer and compiler written in Zig to make compiles faster and get rid of bloat: unnecessary DLL imports, vtables, namespaces, etc.
